# dagster-diagnosis-agent

A diagnostic agent for Dagster.

## Requirements

- Python >=3.12
- uv (https://github.com/astral-sh/uv)

## Setup

Install uv and sync the project environment:

```bash
pip install uv
uv lock && uv sync
```

## Running

Run the agent script:

```bash
uv run dagster-diagnosis-agent
```
